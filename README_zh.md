# EX-FMBE 模拟器

这是一个基于浏览器的轻量级 3D 方块编辑器与模拟器，所有逻辑都集成在一个 `index.html` 文件中。

## 概述

该项目使用 `Three.js` 和浏览器原生模块实现，支持：
- 立方体和旗帜类型方块
- 基于变量的本地和全局变换
- 多选批量编辑
- 颜色预设与自定义贴图导入
- `playanimation` 命令文本的导入与导出
- 撤销/重做、全屏和场景控制

## 功能亮点

- 3D 视图与轨道控制
- 方块选择与多选
- 每个方块的变换变量：位置、旋转、缩放、扩展变换、基准偏移
- 自定义本地变量（`v.xxx`）和全局变量（`t.xxx`）
- 自动增量按钮与滑块调节
- 多选时进入批量编辑模式
- 命令导出支持自定义前缀和后缀
- 导入 `playanimation` 命令文本，自动还原方块
- 可切换轴线、网格、刻度、方块标签和深浅主题

## 运行方式

1. 在浏览器中打开 `index.html`。
2. 如果出现模块导入限制，可使用本地 Web 服务：
   - Python 3: `python -m http.server 8000`
   - Node.js: `npx serve .`
3. 访问 `http://localhost:8000/`。

## 使用说明

- 点击 3D 场景中的方块进行选择。
- Shift/Ctrl 点击可进行多选。
- 使用工具栏按钮来：
  - 导入命令
  - 输出命令
  - 添加材质
  - 切换主题
  - 新建方块
  - 隐藏/显示名称
  - 重置视角
- 在变量面板中编辑数值和表达式。
- 按 `Ctrl+Z` / `Ctrl+Y` 进行撤销/重做。
- 按 `Ctrl+D` 复制选中方块。
- 按 `Delete` / `Backspace` 删除选中方块。
- 按 `F11` 进入全屏。

## 导出 / 导入

- 导出：可自定义命令前后缀，生成每个方块的命令文本。
- 导入：粘贴 `playanimation` 命令文本，编辑器会解析方块数据。
- 如果导入命令包含标记 `'方块类型：旗帜'`，则会自动识别为旗帜类型。

## 说明

- 本模拟器的所有代码集中于 `index.html`。
- `Three.js` 从 CDN 通过 import map 加载。
- 用户界面与编辑逻辑都直接写在该 HTML 文件中。
